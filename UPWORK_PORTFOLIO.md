## Project description
> 600 character max

Built an AI-assisted proposal platform that turns a 30-minute manual quoting process into a 5-minute guided flow — an 85% time saving. Sales reps generate branded, animated proposals from reusable service packages; clients review, e-sign on-screen, pay via Stripe, and download a PDF, all from one link. Includes an admin dashboard for package and terms management, role-based sales-team access, and PostHog funnel analytics. Built with Next.js 15, React 19, Supabase, Tailwind, GSAP, and Stripe.
